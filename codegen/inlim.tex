\hypertarget{link_to_inlim}{}{\Large \bf inlim}

{\it Integrator with (positive) time constant $T$ and non-windup limits on output}

\begin{center}
\includegraphics{codegen/inlim.pdf}
\end{center}

Syntax : \hspace*{10ex}\parbox[t]{\textwidth}{\sf 
\& inlim \\
name of variable $x_i$\\
name of variable $x_j$\\
data name, parameter name or math expression for $T$\\
data name, parameter name or math expression for $x_{min}$\\
data name, parameter name or math expression for $x_{max}$
}

~ \hrule ~

Internal states : none

Discrete variables :  $z \in \{0, 1, -1 \}$

Equations :
\begin{eqnarray}
T \dot{x}_j & = & x_i \;\;\;\;\;\;\;\;\;\; \mbox{if} \;\; z = 0  \label{inlim-1} \\
0 & = & x_j - x_{min} \;\;\;\;\;\;\;\;\;\; \mbox{if} \;\; z = -1  \nonumber \\
0 & = & x_j - x_{max} \;\;\;\;\;\;\;\;\;\; \mbox{if} \;\; z = 1  \nonumber
\end{eqnarray}

\pagebreak
Discrete transitions :

\hspace*{20ex}\begin{minipage}[t]{15cm}
\begin{algorithmic} 
\IF {$z = 0$}
  \IF {$x_j > x_{max}$} 
    \STATE $z \gets 1$
  \ELSIF {$x_j < x_{min}$}
    \STATE $z \leftarrow -1$
  \ENDIF
\ELSIF {$z = 1$}
  \IF {$x_i < 0$}
    \STATE $z \gets 0$
  \ENDIF
\ELSIF {$z = -1$}
  \IF {$x_i > 0$}
    \STATE $z \gets 0$
  \ENDIF
\ENDIF
\end{algorithmic}
\end{minipage}

Initialization of discrete variables :

\hspace*{20ex}\begin{minipage}[t]{15cm}
\begin{algorithmic} 
\IF {$x_j > x_{max}$}
    \STATE {$z \gets 1$}
\ELSIF {$x_j < x_{min}$}
    \STATE {$z \gets -1$}
\ELSE
    \STATE {$z \gets 0$}
\ENDIF
\end{algorithmic}
\end{minipage} 

\vspace*{2ex}
A zero value for $T$ is not allowed. If too small a value is specified for $T$, the solver may encounter a singularity and the simulation may not proceed.
