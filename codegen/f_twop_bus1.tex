\hypertarget{link_to_ftwop1}{}{\Large \bf f\_twop\_bus1}

{\it Similar to f\_inj, computes $f$, an estimate of the frequency (in per unit) at the first bus of a given two-port. Can be used in the model of a two-port only.}

~

Syntax : \hspace*{10ex}\parbox[t]{\textwidth}{\sf 
\& f\_twop\_bus1 \\
name of variable $f$\\
data name, parameter name or math expression for $T$
}

~ \hrule ~

Please refer to f\_inj.
